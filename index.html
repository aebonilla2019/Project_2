The purpose of this project was to analyze the relationship between latitude and climate. The data fof the analysis was retrieved from the OpenWeatherMap API; we retrieved the data for 500 cities that were randomly selected.

To help us us visualize the data we used Matplotib to plot latitude against Max. Temp, Humidity, Cloud %, and Wind. Please feel free to explore this website to find the source data and the visualizations created for the analysis.
